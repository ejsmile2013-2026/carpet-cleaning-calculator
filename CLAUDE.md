# CLAUDE.md — Правила работы с проектом Carpet Cleaning

## Главное правило

Перед любыми изменениями в компонентах или добавлением новых экранов — сначала рассказать,
что планируется сделать и почему. Только после подтверждения — писать код.

---

## Технические правила

- Проект — React SPA. Вся логика в `src/App.tsx`. Не создавать новые компоненты без явного запроса.
- Стек: **React 19 + TypeScript + Vite + Tailwind CSS 4 + motion/react + lucide-react**.
- Не добавлять новые npm-пакеты без обсуждения.
- Не добавлять роутинг и новые страницы — это одностраничный калькулятор.
- Не трогать `vite.config.ts` и `tsconfig.json` без явной необходимости.

---

## Дизайн

- **Фон:** `slate-900` (тёмно-синий)
- **Текст:** белый / `slate-300` для приглушённого
- **Акцентный цвет:** золотой (`gold`, `#c9a227`) — это брендинг клиента, не менять
- Glassmorphism-карточки: `bg-black/30`, `border-white/5`, `backdrop-blur`
- Анимации через `motion/react` — плавные, без резких скачков

---

## Адаптивность

- Вёрстка mobile-first через Tailwind.
- Максимальная ширина основного блока: `max-w-md` (448px).
- Проверять отображение на 375px (iPhone SE) перед тем, как считать задачу готовой.
- Кнопки — минимум 48px высотой (комфортное нажатие пальцем).

---

## Стиль текста

- Язык интерфейса: **английский** (американский клиент).
- Тон: профессиональный, уверенный, без пафоса.
- Короткие подписи, лаконичные лейблы.
